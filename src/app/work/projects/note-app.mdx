## Project Overview
Note App is a focused note‑taking product that keeps capture fast and retrieval simple. It combines authentication, tagging, pinning, and search with a clean UI that works well on mobile.

## Highlights
- JWT auth with persistent sessions.
- Rich note CRUD with pin/unpin and tags.
- Full‑text search for titles and content.
- Dark/light theme and responsive layout.

## Stack
- **Frontend:** React 19, Vite, Tailwind CSS, React Router.
- **Backend:** Node.js, Express, MongoDB (Mongoose), JWT.
